# Movie-Recommended-System

This project is an interactive Movie Recommendation Web App built using Python, Streamlit, and TMDb API.
It recommends movies similar to the one selected by the user based on content-based filtering using similarity scores.

<h3>üöÄ Features</h3>
Recommends top 5 movies similar to a selected movie.
Fetches movie posters and details using TMDb API.
User-friendly Streamlit web interface.
Preprocessed data and similarity scores for fast recommendation results.
Can be deployed easily on Streamlit Community Cloud, AWS, or any cloud platform.

<h3>Tech Stack</h3>
Frontend/UI: Streamlit
Backend Logic: Python
Data Handling: Pandas, NumPy
Similarity Calculation: Scikit-learn (Cosine Similarity)
External API: TMDb (for fetching movie posters)
